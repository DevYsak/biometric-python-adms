# Biometric Test Project

## Step 1: Install Python Package

Open PowerShell or CMD:

py -m pip install pyzk

---

## Step 2: Run Script

Go to project folder:

cd biometric_test_project

Run:

python test_biometric.py

OR

py test_biometric.py

---

## Step 3: Change Device IP

Inside test_biometric.py change:

DEVICE_IP = "192.168.0.141"

to your biometric machine IP.

Default Port:
4370

---

## Expected Output

- Connected Successfully
- User List
- Attendance Logs
